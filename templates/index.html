Search RSS-ATOM Feeds

RSSFeedSearch provides a simple API for searching websites for RSS feeds.

Examples: arstechnica.com, jsonfeed.org, theVerge.com, en.wikipedia.org

Query Parameters

The RSSFeedSearch API accepts the following query parameters:

  • url: The URL to search. Will return 400 Bad Request if not sent.
  • sp: Speed of scraping rss data from web sites. Than slower speed by that more a rss urls scrapied from site. Queried Params: slow - Slower speed, normal or empty - Normal speed, fast - faster speed.
  • result: If query parameter result is equal to json, data showing of a json format, else = data showing in a web interface.
